# proyecto-laravel-api
repositorio de API php laravel

## API de Gestión de Servidores

API RESTful construida con Laravel 12 para gestionar un inventario de servidores.

### Características

- ✅ CRUD completo de servidores
- ✅ Validación de datos con Form Requests
- ✅ Tests automatizados con PHPUnit
- ✅ Análisis estático con PHPStan (nivel 5)
- ✅ Integración con SonarCloud
- ✅ Pipeline CI/CD con GitHub Actions
- ✅ Despliegue automatizado en AWS EC2
- ✅ Dockerizado con Nginx y MySQL
